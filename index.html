JavaScript Utility (CAGR)


function CAGR(begin, end, years) {
  return Math.pow(end / begin, 1 / years) - 1;
}
// Example
console.log("CAGR:", CAGR(1000, 2000, 5));
This helper is used in several dashboards to compute long-term growth quickly.

Financial Formula

Compound Annual Growth Rate (CAGR):

\[ \text{CAGR} = \left( \frac{\text{Ending Value}}{\text{Beginning Value}} \right)^{\frac{1}{\text{Years}}} - 1 \]

We prefer CAGR for smoothing volatility in multi-year comparisons.
